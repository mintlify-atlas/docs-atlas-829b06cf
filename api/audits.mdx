## Create Audit

<RequestExample>
```bash cURL
curl -X POST https://api.latentgeo.com/api/v1/audits/ \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com",
    "market": "US",
    "language": "en",
    "competitors": ["competitor1.com", "competitor2.com"]
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "id": 123,
  "url": "https://example.com",
  "status": "pending",
  "progress": 0,
  "created_at": "2026-03-03T10:00:00Z"
}
```
</ResponseExample>

**POST** `/api/v1/audits/`

Creates a new audit. If competitors and market are provided, the audit pipeline starts immediately. Otherwise, it waits for configuration via chat.

<ParamField body="url" type="string" required>
  Website URL to audit
</ParamField>

<ParamField body="market" type="string">
  Target market (e.g., "US", "UK", "AR")
</ParamField>

<ParamField body="language" type="string">
  Content language (e.g., "en", "es", "fr")
</ParamField>

<ParamField body="competitors" type="array">
  List of competitor domains
</ParamField>

<ResponseField name="id" type="integer">
  Unique audit identifier
</ResponseField>

<ResponseField name="url" type="string">
  Audited website URL
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `pending`, `running`, `completed`, `failed`
</ResponseField>

<ResponseField name="progress" type="integer">
  Progress percentage (0-100)
</ResponseField>

## List Audits

<ParamField query="skip" type="integer" default="0">

</ParamField>

<ParamField query="limit" type="integer" default="100">

</ParamField>

<ResponseField name="audits" type="array">

</ResponseField>

## Run PageSpeed Analysis

**POST** `/api/v1/audits/{audit_id}/pagespeed`

Triggers PageSpeed Insights analysis and returns complete data.

<ParamField query="strategy" type="string" default="both">
  Analysis strategy: `mobile`, `desktop`, or `both`
</ParamField>

<ResponseField name="success" type="boolean">
  Whether analysis completed
</ResponseField>

<ResponseField name="data" type="object">
  Complete PageSpeed data with all metrics, opportunities, diagnostics
</ResponseField>

## Generate PDF Report

**POST** `/api/v1/audits/{audit_id}/generate-pdf`

Generates a comprehensive PDF report with PageSpeed, keywords, backlinks, and GEO analysis.

<ParamField body="force_pagespeed_refresh" type="boolean" default="true">
  Force refresh PageSpeed data
</ParamField>

<ParamField body="force_report_refresh" type="boolean" default="false">
  Force regenerate report markdown
</ParamField>

<ParamField body="force_external_intel_refresh" type="boolean" default="false">
  Force refresh external intelligence data
</ParamField>

<ResponseField name="success" type="boolean">
  Whether PDF generation succeeded
</ResponseField>

<ResponseField name="pdf_path" type="string">
  Supabase storage path (supabase://...)
</ResponseField>

<ResponseField name="file_size" type="integer">
  PDF file size in bytes
</ResponseField>

<ResponseField name="pagespeed_included" type="boolean">
  Whether PageSpeed data was included
</ResponseField>
