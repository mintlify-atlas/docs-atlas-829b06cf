title: Reports

## Get Audit Reports

**GET** `/api/v1/reports/audit/{audit_id}`

Returns all reports associated with an audit.

<ResponseField name="audit_id" type="integer">
  Audit ID
</ResponseField>

<ResponseField name="total_reports" type="integer">
  Number of reports available
</ResponseField>

<ResponseField name="reports" type="array">
  Array of report objects
</ResponseField>

---

## Download Report

**GET** `/api/v1/reports/download/{report_id}`

Redirects to signed Supabase URL for downloading the report file.

<ResponseField name="302" type="redirect">
  Redirects to signed download URL
</ResponseField>

---

## Get Markdown Report

**GET** `/api/v1/reports/markdown/{audit_id}`

Returns the audit report in Markdown format.

<RequestExample>
```bash cURL
curl -X GET https://api.latentgeo.com/api/v1/reports/markdown/123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json
{
  "audit_id": 123,
  "markdown": "# SEO Audit Report\n\n## Executive Summary...\n",
  "created_at": "2026-03-03T10:30:00Z"
}
```
</ResponseExample>

<ResponseField name="audit_id" type="integer">
  Audit ID
</ResponseField>

<ResponseField name="markdown" type="string">
  Full report in Markdown format
</ResponseField>

<ResponseField name="created_at" type="string">
  Report creation timestamp
</ResponseField>

---

## Get JSON Report

**GET** `/api/v1/reports/json/{audit_id}`

Returns the audit report in structured JSON format.

<ResponseField name="audit_id" type="integer">
  Audit ID
</ResponseField>

<ResponseField name="url" type="string">
  Audited URL
</ResponseField>

<ResponseField name="domain" type="string">
  Domain name
</ResponseField>

<ResponseField name="status" type="string">
  Audit status
</ResponseField>

<ResponseField name="is_ymyl" type="boolean">
  Whether site is YMYL (Your Money Your Life)
</ResponseField>

<ResponseField name="category" type="string">
  Site category
</ResponseField>

<ResponseField name="target_audit" type="object">
  Complete audit data object
</ResponseField>

<ResponseField name="external_intelligence" type="object">
  External intelligence data
</ResponseField>

<ResponseField name="search_results" type="object">
  Search engine results data
</ResponseField>

<ResponseField name="fix_plan" type="array">
  Actionable fix plan
</ResponseField>

---

## Generate PDF (Deprecated)

**POST** `/api/v1/reports/generate-pdf`

<Warning>
  This endpoint is deprecated. Use `POST /api/v1/audits/{audit_id}/generate-pdf` instead.
</Warning>

Redirects to the new PDF generation endpoint with HTTP 307.

---

## Error Codes

<ResponseField name="400" type="error">
  Report not ready or invalid request
</ResponseField>

<ResponseField name="404" type="error">
  Report not found
</ResponseField>

<ResponseField name="409" type="error">
  Legacy local PDF paths disabled (Supabase-only mode)
</ResponseField>

<ResponseField name="500" type="error">
  Internal server error
</ResponseField>
