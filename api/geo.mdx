## Citation Tracking

### Start Citation Tracking

**POST** `/api/v1/geo/citation-tracking/start`

Initiates citation tracking for an audit across LLMs.

<ParamField body="audit_id" type="integer" required>
  Audit ID to track
</ParamField>

<ParamField body="industry" type="string" default="general">
  Industry vertical
</ParamField>

<ParamField body="keywords" type="array">
  Keywords to track citations for
</ParamField>

<ParamField body="llm_name" type="string" default="kimi">
  LLM to query (kimi, chatgpt, perplexity)
</ParamField>

---

### Get Citation History

**GET** `/api/v1/geo/citation-tracking/history/{audit_id}`

Returns historical citation data.

<ParamField query="days" type="integer" default="30">
  Number of days of history
</ParamField>

### Get Recent Citations

**GET** `/api/v1/geo/citation-tracking/recent/{audit_id}`
